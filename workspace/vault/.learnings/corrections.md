# Corrections Log

Track moments when the user corrected the agent — these are signals to update MEMORY.md or vault rules.

## Format
- Date: YYYY-MM-DD HH:MM
- What I got wrong: [description]
- Correction: [what user said]
- Root cause: [why I missed it]
- Fix applied: [updated rule/memory/decision]

## Entries
_(Agent appends after each correction)_

# Vault Philosophy — 6 Principles

1. **Everything is a Claim** — File names are assertions, not timestamps
2. **Wiki-Links Build Understanding** — Connect ideas, not folders
3. **Inbox First, Think Later** — Capture raw, process later
4. **Memory > Context** — Use semantic search, not giant prompts
5. **Agent Notes Show Work** — Breadcrumbs help future you
6. **Self-Improving System** — Every error teaches, every win logs
